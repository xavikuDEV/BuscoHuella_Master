# 🐾 BuscoHuella 2026 - Master Repo

Sistema integral para el **Documento Único Animal (DUA)**. Arquitectura de búnker de alta disponibilidad.

## 🏗️ Arquitectura Monorepo

- **apps/web-pro:** Dashboard administrativo y portal público (Next.js 16).
- **apps/mobile-app:** Aplicación nativa para escaneo y gestión (Expo 55).
- **packages/shared-core:** Lógica de negocio, validaciones del DUA y tipos TS.
- **packages/shared-ui:** Librería de componentes comunes (Tailwind).

## 🤖 Sistema de Agentes

Este proyecto es operado por una tríada de agentes:

1. **Orquestador:** Gestión de Roadmap y Calidad.
2. **Architect:** Estructura e Infraestructura.
3. **Specialist:** Ejecución de código (Aider).
